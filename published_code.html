UEMtomaton 2

A Python-driven port of the C++/CLI code used in UEMtomaton. This code is capable of modularly interfacing with delay stage hardwares and is intended for use on the TEM.
License: UEMtomaton is officially licensed by University of Minnesota. More can be read on the GitHub page for UEMtomaton.

ThorLabs Piezocontroller Python Interface

A Python-driven interface used to control piezocontrollers sold by ThorLas.
License: The license for this software is under the ThorLabs open software license.

Pulsed Ponderomotive Phase Plate

Companion scripts to the papers published in Cell Reports Methods, 2023. A forward-stepping finite element model of a pulsed ponderomotive phase plate coded in Python with numba support for faster computation times. A GUI using PyQt5 has been included for user-friendliness.
License: The license for this simulation software is officially handled by Columbia University.

UEMtomaton

A companion script linked to the paper published in Ultramicroscopy, 2021. An automation software coded in C++/CLI Windows Forms to enable efficient acquisition of ultrafast electron microscope images. The program is designed to present the setup and completion of these experiments in a user-friendly manner by communicating between two hardware components with Berkeley sockets and Gatan Digital Micrograph.
License: UEMtomaton is officially licensed by University of Minnesota. More can be read on the GitHub page.

Contrast Through Specimen

A series of softwares that aid in the analysis of UEM scans and simulate the contrast expected from passing electrons through specimens (primarily Germanium).
Germanium Two-Temperature Model:
A two-temperature model of germanium used to determine the peak temperature picoseconds after photoexcitation. Uses a finite-element method.
Lamb Wave Contrast:
A companion script to the paper published in Structural Dynamics, 2020. An electron dynamical multi-slice scattering script used to correlate the simulated strain of a Lamb wave to the bright-field contrast observed in ultrafast electron microscopy.
Ultrafast Electron Microscopy Analysis Suite:
Image analysis tools dedicated to the rapid automated processing of ultrafast electron microscopy scans. Such tools include background subtraction, space-time-plot formation, region-of-interest selection, and various other tools and scripts.
License: GermaniumTTM, LambWaveContrast, and UEM Analysis Suite is officially licensed by University of Minnesota. More can be read on the GitHub page.
